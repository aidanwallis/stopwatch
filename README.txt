A simple Python stopwatch app made using flet

Authors:
Aidan Wallis
Reign Pierson

Features:
Stopwatch with dynamically changing start/stop button
Save Time button that allows you to save the current value on the timer for comparison
Lap button to save the time on lap completion and display it in a numbered list with other lap times

Usage:
Install flet and its dependencies (https://flet.dev/docs/)
Run gui.py
